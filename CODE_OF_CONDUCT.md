# Code of Conduct

## Our Standard

本项目希望建立一个尊重、开放、可靠的医学本科资料协作环境。

请做到：

- 尊重不同年级、专业方向和学习背景的同学。
- 对资料质量、来源和授权保持诚实。
- 批评内容，不攻击个人。
- 不发布患者隐私、教师个人隐私或机构敏感信息。
- 不把学习资料当作医疗建议传播。

## Unacceptable Behavior

不可接受的行为包括：

- 人身攻击、骚扰、歧视或羞辱。
- 发布盗版资料、内部资料或未经授权内容。
- 发布含隐私或敏感信息的实习、病例或考试材料。
- 明知内容错误仍拒绝修正。

## Enforcement

维护者可以编辑、关闭或拒绝违反本规范的 issue、PR 或讨论。严重或重复违反者可能会被限制参与项目。

# Freshman / 大一

大一资料主要服务于医学本科起步阶段：适应大学学习节奏，建立基础课程学习方法，整理可公开分享的入门资料。

## 分类入口

| 分类 | 入口 | 适合内容 |
|---|---|---|
| 总结 | [summaries](summaries/) | 基础课程总结、复习框架、学习方法整理。 |
| 经验贴 | [experience-posts](experience-posts/) | 适应大学学习、选课、时间管理、考试经验。 |
| 文档 | [documents](documents/) | 原创或授权的课程整理、公开资源索引、PDF/Word 说明。 |

## 推荐主题

- 医学本科起步经验。
- 基础课程预习和复习方法。
- 解剖、生理、生化等课程的学习框架。
- 从高中学习方式过渡到医学学习方式的经验。

## 投稿提示

新增资料请优先使用 [templates](../../templates/) 中的模板。不要上传教材扫描件、内部课件、未授权题库或含隐私的信息。

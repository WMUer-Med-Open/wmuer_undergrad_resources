# Fifth-Year Internship / 大五与实习

大五与实习资料主要服务于临床轮转、执业考试准备和毕业前能力整合。这个阶段的内容最容易涉及隐私，投稿时必须格外谨慎。

## 分类入口

| 分类 | 入口 | 适合内容 |
|---|---|---|
| 总结 | [summaries](summaries/) | 轮转复习框架、执业考试整理、临床能力清单。 |
| 经验贴 | [experience-posts](experience-posts/) | 科室轮转经验、实习节奏、执考准备经验。 |
| 文档 | [documents](documents/) | 原创或授权的轮转资料索引、公开学习文档、PDF/Word 说明。 |

## 推荐主题

- 各科室轮转学习重点。
- 实习记录、技能训练和复盘方法。
- 执业考试复习节奏和资料使用经验。
- 从学生到临床工作场景的过渡经验。

## 投稿提示

严禁上传患者信息、病历截图、检查报告、排班、群聊、科室内部流程或任何可识别个人的信息。经验贴应写成可复用的方法，而不是暴露具体病例或科室细节。

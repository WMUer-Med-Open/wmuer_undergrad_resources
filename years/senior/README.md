# Senior / 大四

大四资料主要服务于临床课程、见习和考试阶段：把课程知识、病例思维和实操经验放在同一学习框架里。

## 分类入口

| 分类 | 入口 | 适合内容 |
|---|---|---|
| 总结 | [summaries](summaries/) | 临床课程总结、常见病复习框架、诊断治疗原则整理。 |
| 经验贴 | [experience-posts](experience-posts/) | 见习经验、考试经验、临床课程学习策略。 |
| 文档 | [documents](documents/) | 原创或授权的临床课程文档、见习资料索引、PDF/Word 说明。 |

## 推荐主题

- 内外妇儿等临床课程复习框架。
- 常见症状、常见病、鉴别诊断的学习方法。
- 见习中的记录方式和隐私边界。
- 临床考试、技能训练和病例讨论经验。

## 投稿提示

见习相关内容必须去除患者、教师、科室和医院内部非公开信息。不要上传病历、检查报告、排班或群聊截图。

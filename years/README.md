# Years

资料库主体按年级组织。每个年级下固定包含 `summaries/`、`experience-posts/` 和 `documents/` 三类内容。

## 年级索引

| 年级 | 阶段定位 | 入口 |
|---|---|---|
| 大一 | 适应医学本科节奏，建立基础学习方法 | [freshman](freshman/) |
| 大二 | 基础医学核心课程，形成机制理解 | [sophomore](sophomore/) |
| 大三 | 桥接基础与临床，开始系统化疾病学习 | [junior](junior/) |
| 大四 | 临床课程、见习、考试和病例思维 | [senior](senior/) |
| 大五/实习 | 临床实习、轮转经验、执业考试准备 | [fifth-year-internship](fifth-year-internship/) |

## 固定分类

| 分类 | 内容 |
|---|---|
| `summaries/` | 课程总结、系统总结、疾病学习总结、复习提纲。 |
| `experience-posts/` | 学习经验、考试经验、教改班经验、实习经验。 |
| `documents/` | 可公开分享的 Markdown、PDF、Word 文档及其说明。 |

## 维护原则

- 新资料先放入最匹配的年级，不确定时写明适用范围。
- 不建议一开始拆出太多子目录，优先维护 README 索引表。
- PDF/Word 文件必须放在 `documents/`，并配套同名 Markdown 说明。
- 临床相关内容必须标注更新时间和局限。

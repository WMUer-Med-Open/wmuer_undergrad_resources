# Junior / 大三

大三资料主要服务于基础到临床的过渡阶段：开始围绕系统、疾病和诊断思路组织知识。

## 分类入口

| 分类 | 入口 | 适合内容 |
|---|---|---|
| 总结 | [summaries](summaries/) | 系统课程总结、疾病学习框架、诊断思路整理。 |
| 经验贴 | [experience-posts](experience-posts/) | 临床课程入门、考试准备、学习路径经验。 |
| 文档 | [documents](documents/) | 原创或授权的系统复习文档、病例学习索引、PDF/Word 说明。 |

## 推荐主题

- 系统课程和疾病学习框架。
- 基础机制到临床表现的衔接。
- 常见病诊断思路和鉴别诊断学习方法。
- 临床课程初期的资料选择和复习节奏。

# Files

这里用于说明和索引合规的 PDF、Word 补充资料。

本仓库仍以 Markdown 为主要组织方式。PDF 和 Word 文件只作为补充资料，必须满足：

- 原创、明确授权或可公开再分发。
- 不包含盗版教材、教材扫描件、内部课件、未授权题库或原始试卷扫描件。
- 不包含患者隐私、教师个人隐私、科室敏感信息或医院内部非公开流程。
- 文件名使用小写英文、数字和连字符。
- 必须配套 Markdown 索引说明来源、授权、适用阶段和使用建议。

建议目录：

```text
files/
  pdf/
  word/
```

上传具体文件前，请先在对应目录补充索引 README 或资源条目。

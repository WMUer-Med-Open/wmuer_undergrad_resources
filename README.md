# WMUer Undergraduate Medical Resources

> 面向医学本科生的开放资料库，按年级整理总结、经验贴和文档。  
> An open undergraduate medical resource library organized by academic year.

## 中文简介

`wmuer_undergrad_resources` 是一个面向医学本科生的开放资料库。它不做复杂的学科分叉，先按年级建立清晰入口，让同学能直接找到对应阶段的总结、经验贴和文档。

资料整理仍然鼓励 **按照疾病的学习模式进行整合与分享**：在每个年级的总结和文档中，把基础医学机制、病理生理、临床表现、诊断思路、治疗原则、考试重点和实习经验联系起来，帮助同学从“课程章节”走向“临床问题”。

本仓库支持 Markdown、PDF 和 Word 文件。PDF/Word 必须是原创、明确授权或可公开再分发的资料，并建议配套 Markdown 说明来源、授权、适用阶段和用途。不要上传教材扫描件、学校内部课件、未授权题库、录音录像或任何含隐私的信息。

## English Overview

`wmuer_undergrad_resources` is an open resource library for undergraduate medical students. The repository is organized by academic year so that students can find summaries, experience posts, and documents for their current stage.

The project still encourages a **disease-based learning model** inside each year: connect basic mechanisms, pathophysiology, clinical manifestations, diagnostic reasoning, treatment principles, exam review, and clerkship experience around diseases or clinical problems.

The repository supports Markdown, PDF, and Word files. PDF/Word files must be original, clearly licensed, or publicly redistributable, and should be accompanied by Markdown notes explaining source, license, target stage, and intended use. Scanned textbooks, internal courseware, unauthorized question banks, recordings, videos, and privacy-sensitive materials are not accepted.

## Contribution Principles

- 优先提交 Markdown；PDF/Word 可作为补充资料提交。
- 每个文件放入对应年级的 `summaries`、`experience-posts` 或 `documents`。
- 文件名使用小写英文、数字和连字符，例如 `internal-medicine-review.md`。
- 必须说明资料来源、授权状态、适用年级和是否原创。
- 不得提交盗版教材、内部课件、未授权题库、原始试卷扫描件或含患者/教师/科室隐私的信息。
- 医学内容应尽量标注更新时间，避免把过时经验写成固定结论。

## Medical Disclaimer

本资料库仅用于医学教育、课程学习和同学间经验交流，不构成医疗建议，不能替代执业医师判断、学校课程要求、医院规章制度或官方指南。任何临床相关内容都应结合最新指南、教师要求和所在机构规范进行判断。

This repository is for medical education, coursework, and peer learning only. It does not provide medical advice and must not replace licensed clinical judgment, institutional requirements, school curricula, or official guidelines.

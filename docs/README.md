# Documentation

## 文档索引

| 文档 | 用途 |
|---|---|
| [content-standard.md](content-standard.md) | 判断一份资料是否足够清晰、可靠、可维护。 |

| [maintainer-guide.md](maintainer-guide.md) | 给维护者使用的审核、整理和清理流程。 |

## 推荐阅读顺序

投稿者建议先看 [CONTRIBUTING.md](../CONTRIBUTING.md)，再按需要阅读这里的细则。维护者建议先看 [maintainer-guide.md](maintainer-guide.md)。

# Copyright And Privacy

医学资料库的底线是版权清楚、隐私安全、医学表述克制。任何不确定是否能公开的内容，都应该先不上传。

## 可以收录

- 自己原创的总结、经验贴和学习框架。
- 已获得作者或版权方明确授权的资料。
- 明确允许再分发的开放资料。
- 只做索引、介绍和使用建议的公开资源链接。
- 少量、必要、合规引用，并注明来源。

## 不可以收录

- 盗版教材、教材扫描件、翻拍资料。
- 学校内部课件、内部讲义、未授权题库、原始试卷扫描件。
- 付费课程资料、网课讲义、录音录像。
- 没有来源、没有授权说明的网盘文件。
- 患者、教师、同学、医护人员的隐私信息。
- 医院或科室内部非公开流程、账号、排班、群聊或联系方式。

## PDF/Word 授权判断

提交 PDF/Word 前，请至少能回答：

| 问题 | 必须明确吗 |
|---|---|
| 这个文件是谁制作的？ | 是 |
| 制作者是否允许公开？ | 是 |
| 是否允许复制和再分发？ | 是 |
| 是否包含教材、课件、题库或试卷内容？ | 是 |
| 是否包含任何可识别个人的信息？ | 是 |

如果任何一项无法确认，不要上传文件本体。可以改为写一份 Markdown 索引，说明资料在哪里公开获得、如何判断是否适合使用。

## 隐私去标识化

实习、见习、病例讨论相关内容尤其要谨慎。提交前请删除：

- 姓名、电话、住址、身份证号、学号、工号。
- 床号、住院号、门诊号、影像号、病历号、检查号。
- 精确日期、精确地点、罕见病程细节等组合后可识别身份的信息。
- 检查图片、报告截图、聊天截图、排班截图。
- 教师或医护人员的私人评价、联系方式和非公开安排。

## 医学表述边界

本仓库内容只能用于学习交流。请避免：

- 对具体患者给出诊断或治疗建议。
- 把个人实习经验写成临床规范。
- 使用“必须这样处理”“唯一方案”等绝对化表述。
- 省略更新时间而陈述临床结论。

推荐表述：

- “课程复习中可按以下框架理解。”
- “个人经验，适用于当时课程安排。”
- “临床相关内容需结合最新指南和所在机构规范核对。”
- “以下仅为学习笔记，不构成医疗建议。”

## 发现问题怎么办

如果发现仓库中有疑似版权或隐私问题，请创建 Issue，选择版权/隐私报告模板。维护者应优先处理这类问题，必要时先移除文件再进一步核实。

# Templates

这里存放投稿模板。复制模板到对应年级目录后，再修改标题、元信息和正文。

## 模板索引

| 模板 | 用途 | 推荐目录 |
|---|---|---|
| [summary-template.md](summary-template.md) | 课程总结、系统总结、疾病学习总结、复习框架 | `years/<year>/summaries/` |
| [experience-post-template.md](experience-post-template.md) | 学习经验、考试经验、见习或实习经验 | `years/<year>/experience-posts/` |
| [document-entry-template.md](document-entry-template.md) | PDF/Word/Markdown 文档说明 | `years/<year>/documents/` |

## 使用方式

1. 复制最匹配的模板。
2. 文件名改成小写英文、数字和连字符，例如 `cardiovascular-review.md`。
3. 填写 `title`、`year`、`category`、`author`、`license`、`last_updated` 和 `source`。
4. 在对应分类 README 的资源索引表中增加一行。

PDF/Word 文件不能只上传文件本体，必须配套同名 Markdown 说明。
